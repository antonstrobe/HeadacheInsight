# Repo Instructions

- After any change in this repository, validate the result by building the Android app before final handoff.
- After any change in this repository, install the updated Android APK on the connected phone via ADB before final handoff.
- After any change in this repository, commit only the relevant project files and push them to `origin`.
- Prefer the existing Windows helper flow in `scripts/build-headacheinsight.ps1` or the equivalent direct Gradle + ADB commands when running the build/install step.
- If build, phone install, commit, or push is blocked by an external issue, report the blocker clearly in the final message.
