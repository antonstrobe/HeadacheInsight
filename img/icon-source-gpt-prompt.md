# GPT Prompt For App Icon

Нужен исходный мастер-файл для иконки Android-приложения `HeadacheInsight`.

## Что нужно сгенерировать

- квадратный `1024x1024` PNG
- прозрачный фон
- аккуратный alpha channel
- один крупный центральный символ
- без текста, без букв, без цифр, без рамки
- стиль: premium, calm, medical-tech, low-stimulation

## Recommended Prompt

```text
Create a premium mobile app icon master artwork for an Android health app called HeadacheInsight.

Output requirements:
- 1024x1024
- transparent background
- PNG-ready composition with clean alpha channel
- one centered symbol only
- no text, no letters, no watermark, no mockup, no phone frame
- keep generous transparent padding around the symbol
- crisp silhouette, readable at small sizes

Visual direction:
- low-stimulation, calm, refined, trustworthy
- not playful, not childish, not loud
- subtle medical-tech feel without looking like a hospital logo
- modern, slightly glossy or softly layered vector-like finish

Main symbol:
- an abstract rounded head silhouette or calm circular emblem
- a focused accent near the temple/forehead area to suggest headache tracking
- a soft insight/data signal integrated into the mark
- the symbol must feel supportive, clear, and premium

Color palette:
- deep midnight navy
- muted dark teal
- restrained crimson or warm red accent for the pain focus area
- optional soft violet only as a very small secondary accent
- avoid bright white backgrounds

Composition constraints:
- icon must work on transparent background
- transparent outer area must remain truly transparent
- no full square background tile
- no tiny details
- no realistic anatomy
- no skulls, brains, blood, lightning bolts, pills, syringes, or horror elements
- no busy gradients

Preferred look:
- balanced geometry
- soft rounded edges
- premium Android launcher icon style
- suitable for a privacy-first migraine and headache journal app

Negative constraints:
- no text
- no photorealism
- no mascots
- no clutter
- no harsh neon
- no flat corporate stock logo
```

## Short Version

```text
Create a 1024x1024 transparent PNG app icon master for a calm premium migraine-tracking app. One centered abstract symbol, deep navy + muted teal + restrained red accent near the temple, low-stimulation, readable at small size, clean alpha, no text, no background tile, no realistic anatomy, no clutter.
```

## Что положить в репозиторий

Сохраните итоговый файл как:
